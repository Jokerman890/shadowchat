# Design Tokens

## Farbe

- Base Dark
- Blue
- Indigo
- Violet
- Cyan
- Emerald
- Liquid Light: Porcelain, Lavender Mist, Ice Blue, Blush, Deep Text, Soft Text

## Gradient

- Core Aurora
- Ice Signal
- Secure Trust
- Night Depth
- Messenger Accent: Lila-Blau-Verlauf fuer primaere Akzente, Unread-Badges und fokussierte Controls

## Material

- base
- frosted
- liquid
- hero
- glass panel
- floating card
- message bubble

## Motion

- xs
- s
- m
- l
- xl
- press scale
- selected scale
- state transition
- screen transition

## Regeln

- Klare Token-Namen.
- Keine Screen-spezifischen Einzelwerte im Produktkern.
- Reduce Motion und Reduce Transparency werden mitgedacht.
- Liquid-Glass-Flaechen bleiben hell, lesbar und subtil; Transparenz darf keine Texte destabilisieren.
- Motion-Tokens fuer Press Scale, Selected Scale, State Transition und Screen Transition erklaeren Fokus und Status ohne permanente Unruhe.
