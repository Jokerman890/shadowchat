# ADR-0001 Monorepo

## Entscheidung
ShadowChat wird als Monorepo organisiert.

## Begründung
- gemeinsamer Core
- gemeinsame Dokumentation
- gemeinsame CI
- koordinierte Änderungen an Core und Mobile-Apps

## Konsequenzen
- klare Modulgrenzen sind Pflicht
- Build-Systeme müssen sauber getrennt werden
