# ADR-0003 Native UI

## Entscheidung
Die Apps werden nativ gebaut: iOS mit SwiftUI, Android mit Jetpack Compose.

## Begründung
- beste Systemintegration
- starke Motion- und Material-Umsetzung
- saubere Plattform-UX

## Konsequenzen
- gemeinsame Produktlogik bleibt im Core
- Plattform-Teams brauchen klare UI-Grenzen
