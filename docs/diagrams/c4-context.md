# C4 Context

## System
ShadowChat Mobile

## Akteure
- Endnutzer
- optional Team / Business-Nutzer

## Externe Systeme
- Homeserver / Auth-System
- Push-Dienste iOS / Android
- Billing-Plattformen
- optionale externe Netzwerke für spätere Bridge-Szenarien

## Zweck
ShadowChat stellt sichere, moderne Mobile-Kommunikation bereit und verbindet Chat, Session, Security und Produktlogik in einem klar getrennten Systemkontext.
