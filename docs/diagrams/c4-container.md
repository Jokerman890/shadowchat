# C4 Container

## Container
- Mobile App iOS
- Mobile App Android
- gemeinsamer Core
- lokaler Storage / Session-Daten
- Notification Handling
- Billing / Entitlements
- Security Center

## Grundregel
UI, Domain, Security, Notifications und spätere Bridge- oder Billing-Themen bleiben klar getrennte Container- oder Modulgrenzen, auch wenn sie im selben Produkt auftreten.
