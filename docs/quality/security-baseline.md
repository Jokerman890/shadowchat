# Security Baseline

## Grundsatz
ShadowChat wird als sicherheitsorientiertes Produkt entwickelt. Sicherheit ist Kernanforderung, kein Nachtrag.

## Mindestanforderungen
- sichere lokale Speicherung
- saubere Session-Trennung
- kein unnötiges Logging von Inhalten
- sichtbare Device- und Trust-Zustände
- Reduce-Motion und High-Contrast berücksichtigen Nutzersicherheit und Bedienbarkeit

## Produktregeln
- native und externe Kontexte werden getrennt markiert
- Security Center ist Teil des Produkts
- sensible Flows erhalten eigene Tests
