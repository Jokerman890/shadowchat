# Release Definition

## MVP ist releasefähig, wenn
- Login stabil funktioniert
- Chat-Liste produktionsreif ist
- Chat-Raum produktionsreif ist
- Nachrichtenfluss zuverlässig ist
- Push-Open-Flow stabil ist
- Security-Grundlagen sichtbar sind
- Free / Pro-Struktur im Modell vorbereitet ist
- Reduce Motion sauber funktioniert

## Nicht releasefähig, wenn
- Kernflüsse abstürzen
- Push den falschen Kontext öffnet
- Security-Zustände unklar sind
- Motion Lesbarkeit oder Performance spürbar schädigt
