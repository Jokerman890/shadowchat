# ShadowChat MVP Scope

## Ziel
Ein produktionsnaher Premium-Messenger-Kern für iOS und Android mit starker Chat-UX und sauberer technischer Basis.

## In Scope
- Onboarding
- Login / Session Start
- Chat-Liste
- 1:1-Chats
- Gruppenchat-Basis
- Textnachrichten
- Medien-Basis
- Reply
- Reactions
- Sent / Delivered / Read States
- Suche-Basis
- Push-Basis
- Security Center Basis
- Entitlement-Modell vorbereitet

## Out of Scope
- umfassende Bridge-Funktionen
- ausgebaute Business-Administration
- komplexe Call-Szenarien
- stark erweiterte Personalisierung
- Desktop / Web

## Release-Kriterien
- Login stabil
- Chat-Liste und Chat-Raum produktionsreif
- Nachrichtenfluss zuverlässig
- Push-Open-Flow stabil
- Security-Grundlagen sichtbar
- Motion reduziert bei Reduce-Motion-Modus
