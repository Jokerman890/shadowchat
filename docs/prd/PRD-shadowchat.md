# ShadowChat PRD

Version: 0.1
Status: Draft
Plattformen: iOS, Android

## Vision
ShadowChat ist ein hochwertiger, sicherheitsorientierter Messenger mit moderner visueller Identität, nativen Mobile-Apps und vorbereitetem Abo-Modell.

## Zielgruppen
- Privatnutzer mit hohem Anspruch an UX und Security
- Power User mit Bedarf an Kontrolle und Multi-Account
- später Teams und Business-Nutzer

## Kernversprechen
- Premium Chat-Erlebnis
- klare Security- und Trust-Kommunikation
- Free / Pro / Business vorbereitet
- Bridges nur als optionale Erweiterung

## MVP
- Onboarding und Login
- Chat-Liste
- 1:1-Chat
- Gruppenchat-Basis
- Text, Medien-Basis, Reactions, Reply
- Push-Basis
- Security Center Basis

## V1
- Calls-Basis
- sichtbarer Multi-Account
- Subscription Screens
- Share Extension iOS

## V2
- Bridge Hub
- Business-Funktionen
- Advanced Privacy Controls
